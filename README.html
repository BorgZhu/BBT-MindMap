Beschreibung

Ziel dieses Programmes ist es, dass man schnell und einfach MindMaps erstellen kann. Die Erstellung soll intuitiv, nicht überladen und leicht verständlich sein, jedoch sollten alle Funktionalitäten enthalten sein, die notwendig sind um möglichst viele einsatzgebiete abzudecken. Wir haben uns auf das Wesentliche konzentriert, sodass verschiedene Funktionalitäten, die man aus anderen Programmen kennt entfallen sind. Zum Beispiel kann man die den Verbindungen nicht unterschiedliche Styles geben, des weiteren kann man keine Bilder als Knoten hinzufügen auch entfallen sind Funktionen wie Touch-Support und Gestenerkennung.

unterstützte Funktionalitäten

Das Programm unterstützt verschiedene Formen von Knoten, sowie das hinzufügen von Icons um den Text, den man in die Knoten eintragen kann, zu untermalen. Weiter Funktionen sind:

  • Textgröße verändern
  • export von XML
  • import von XML
  • export als PNG
